# AI Warehouse & Logistics Command Center

## Overview

The AI Warehouse & Logistics Command Center is a portfolio project that simulates an intelligent logistics control tower for an e-commerce company.

It analyzes orders, warehouse capacity, inventory availability, courier performance, delivery risk, and logistics exceptions to help operations leaders identify issues before they impact customers.

## Business Problem

E-commerce companies often struggle with late deliveries, warehouse congestion, inventory shortages, courier delays, and manual exception handling.

These problems can lead to poor customer experience, revenue loss, SLA breaches, and operational inefficiency.

## Solution

This project uses Python, FastAPI, pandas, Streamlit, n8n, Claude, and dashboard-ready outputs to simulate a logistics intelligence system.

The system will:

- Load order data
- Check inventory availability
- Recommend the best warehouse
- Predict delivery delay risk
- Classify logistics exceptions
- Calculate executive KPIs
- Export reports
- Display insights in a Streamlit dashboard
- Send exception alerts through n8n
- Generate executive summaries using Claude

## Portfolio Story

This project demonstrates how AI and automation can help logistics teams move from reactive firefighting to proactive exception management.

Instead of simply tracking what already happened, the system helps identify risk early, prioritize exceptions, and support faster operational decision-making.
